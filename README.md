
# 🌲 Forest Monitoring System

## Description
The Forest Monitoring System is a web-based application developed to monitor and manage forest-related information efficiently. It provides a centralized platform for maintaining forest records, tracking environmental data, and supporting sustainable forest management.

## Features
- Forest data management
- User authentication
- Real-time monitoring
- Report generation
- Secure data storage

## Technologies
- Python
- Flask
- HTML
- CSS
- JavaScript
- MySQL
